h1,h2,h3,h4,h5,h6要素　

説明
h1,h2,h3,h4,h5,h6要素はセクションの見出しを表すために使う要素です。 これらの要素のセマンティクスとしての意味は「見出しとセクション」です。 これらの要素は名前に含まれる数字によってランクが決められています。数字が小さいほど描画される文字は大きくなります。よってh1が一番ランクが高く文字のサイズは一番大きくなります。逆にh6はこれらの要素の中で一番ランクが低く文字のサイズは一番小さくなります。
マークアップ例（HTMLコード）
<h1>これはh1要素です。</h1>
<h2>これはh2要素です。</h2>
<h3>これはh3要素です。</h3>
<h4>これはh4要素です。</h4>
<h5>これはh5要素です。</h5>
<h6>これはh6要素です。</h6>
動作結果

これはh1要素です。

これはh2要素です。

これはh3要素です。

これはh4要素です。

これはh5要素です。
これはh6要素です。

a要素　

説明
a要素は要素の内容に対してハイパーリンク（ハイパーテキストアンカー）を与えることができます。href属性でURLを与えることによって、内容をクリックした場合のそのURLにアクセスすることができます。a要素がhref属性を持たない場合はその要素はプレースホルダーを表します。
マークアップ例（HTMLコード）
<a href="https://developer.mozilla.org/ja/docs/Web">この文章をクリックするとMDN Web Docdsへアクセスできます。</a>
動作結果
この文章をクリックするとMDN Web Docdsへアクセスできます。

abbr要素

説明
abbr要素は略語や頭文字を表すために使われる要素です。カーソルを合わせることで略語を表示することができます。オプションで略語を展開することもできます。title属性を利用することで略語の展開を指定できます。
マークアップ例（HTMLコード）
<abbr title="Uniform Resource Locator">URL</abbr>とはUniform Resource Locatorの略語です。
この文書は<abbr title="HyperText Markup Language">HTML</abbr>で作られています。
動作結果
URLとはUniform Resource Locatorの略語です。
この文書はHTMLで作られています。
